# Customs Bot

A feature-rich Discord bot designed for managing gaming tournaments in a university server. This project demonstrates advanced Python programming, database integration, and Discord API usage, making it an excellent addition to any developer's portfolio.

## Features

- **Player and Team Management**: Register players, create and manage teams, and assign roles.
- **Automated Tournament Brackets**: Organize matches, generate fair pairings, and progress through tournament rounds.
- **Leaderboard System**: Track player and team rankings with a dynamic leaderboard.
- **ELO-Based Matchmaking**: Match logic inspired by chess ELO, factoring in both individual and team strength for fair competition.
- **Role-Based Permissions**: Admin-only commands for sensitive operations, ensuring secure and organized management.
- **Database Integration**: Persistent storage of players, teams, matches, and results using a custom database schema.
- **Custom Utilities**: Includes utilities for ELO calculations, Discord embed formatting, and more.

## Technical Highlights

- **Python**: Core logic and Discord bot implementation.
- **Discord.py**: For seamless Discord integration and command handling.
- **Modular Design**: Organized with cogs for scalability and maintainability.
- **Database Layer**: Custom models and queries for robust data management.
- **Utils**: Helper modules for ELO, embeds, and other reusable logic.

## Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Configure your Discord bot token and database settings**
4. **Run the bot**:
   ```bash
   python main.py
   ```
